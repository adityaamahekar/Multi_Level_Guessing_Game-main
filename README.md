# 🎯 Multi-Level Guessing Game

## 📌 Overview
Multi-Level Guessing Game is an interactive web-based game where users try to guess a randomly generated number. The game provides hints, multiple difficulty levels, and a scoring system to make gameplay engaging and challenging.

---

## 🚀 Features

### 🔹 Core Features
- Random number generation
- User input for guessing
- Hint system (Too High / Too Low)
- Result display (Win / Lose)

### 🔹 Advanced Features
- Difficulty levels (Easy, Medium, Hard)
- Limited number of attempts
- Score calculation based on performance
- Replay game functionality

### 🌟 Creative Feature
- Performance Badge System (e.g., Smart Guesser, Lucky Player)
- Best Score stored using Local Storage

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage


---

## ⚙️ How It Works
1. User selects a difficulty level.
2. System generates a random number based on difficulty.
3. User enters a guess.
4. System gives hints (Too High / Too Low).
5. Game ends when:
   - User guesses correctly → Win
   - Attempts finish → Lose
6. Score is calculated based on remaining attempts.

---

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
3. Select difficulty and start playing
